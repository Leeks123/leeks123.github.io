Swift 간단 정리 1

17.08.2020 — Swift — 1 min read

인프런 "야곰의 스위프트 기본문법 강좌"를 정리했습니다.

상수와 변수

  • 상수선언 키워드 let
    • 상수선언 let 이름: 타입 = 값
  • 변수선언 키워드 var
    • 변수 선언 var 이름: 타입 = 값

기본 데이터 타입

  • Bool : true, false. 0,1로 대체할 수는 없다
  • Int : 정수
  • Uint : 양의 정수
  • Float : 32비트, 부동소수형
  • Double : 64비트, 부동소수형
  • Character : 문자. 유니코드. 큰따옴표
  • String : 문자열. 큰따옴표.

다른 데이터 타입과 자료교환은 암시적으로 불가능

Any, AnyObject, nil

  • Any : 스위프트의 모든 타입을 지칭하는 키워드
  • AnyObject : 모든 클래스 타입을 지칭하는 프로토콜
  • nil : 없음을 의미하는 키워드. Any, AnyObject에 할당불가

스위프트는 함수형 패러다임을 포함하는 다중 패러다임 언어

스위프트의 함수는 일급객체이므로 변수, 상수 등에 저장이 가능하고 매개변수로 전달도 가능

조건문

if else

  • 소괄호는 생략가능하지만 중괄호는 생략 불가
  • 조건문은 항상 Bool 타입이 와야 함
1if condition {
2  ...
3} else {
4  ...
5}

switch

  • Default는 반드시 작성할 것
  • break 문을 사용하지 않음. 굳이 사용하겠다면 fallthrough

옵셔널

값이 있을 수도 있고 없을 수도 있는 것. nil 가능성을 문서화 하지 않아도 코드만으로 충분히 표현가능. 옵셔널이 아니라면 nil 체크를 안 해도 안심

옵셔널은 열거형 + 제너럴의 형태

1enum Optional<Wrapped> : ExpressibleByNilLiteral {
2  case none
3  case some(Wrapped)
4}
5
6let optionalValue: Optional<Int> = nil
7let optionalValue: Int? = nil
!
1var optionaleVaue: Int! = 100
2
3switch optionalValue { 
4  case .none:
5    print("This Optional variable is nill")
6  case .some(let value):
7    print("Value is \(value)")
8}
  • Inmplicitly Unwrapped Optional (암시적 추출 옵셔널)
  • 기존 변수처럼 사용 가능
  • nil 할당 가능
?
  • nil 할당 가능
  • 기존 변수처럼 사용 불가 - 옵셔널과 일반 값은 다른 타입이므로 연산 불가

Optional Unwrapping

Optional Binding
1func printName(name: String) {
2  print(name)
3}
4var myName: String! = nil
5
6if let name: String = myName {
7  printName(name)
8} else {
9  print("myName == nil")
10}

if-let 을 이용하여 name 에 myName 을 바인딩하여 사용 name 은 if-let 구문 내에서만 사용 가능

Force Unwrapping

옵셔널의 값을 강제로 추출. 비추 방식

1var myName2: String? = "optional"
2printName(myName2!) // !를 붙어 강제 추출
3
4myName2 = nil
5print(myName!)
6// 강제추출 시 값이 없으므로 런타임 오류 발생
7
8var yourName: String! = nil
9printName(yourName)
10// 암시적 추출 옵셔널을 붙이면 알아서 변수뒤에 !를 붙임
11// nil값이 전달되기 떄문에 런타임 오류 발생

구조체

1struct Sample {
2  var mutableProperty: Int = 100 // 가변 프로퍼티
3  let immutableProperty: Int = 100 // 불변 프로퍼티
4  
5  static var typeProperty: Int = 100 // 타입 프로퍼티
6  
7  // 인스턴스 메서드
8  func instanceMethod() {
9    print("instance method")
10  }
11  // 타입 메서드
12  static func typeMethod() {
13    print("type method")
14  }
15}
16
17// 가변 인스턴스, 가변프로퍼티만 변경 가능
18var mutable: Sample = Sample()
19
20// 불변 인스턴스, 어떤 프로퍼티도 변경 불가
21let immutable: Sample = Sample()
22
23Sample.typeMethod() // type method

클래스

  • 구조체와 굉장히 유사하지만 구조체는 값 타입이고, 클래스는 참조 타입이다
  • 다중 상속이 되지 않음
1class Sample {
2  var mutableProperty: Int = 100 // 가변 프로퍼티
3  let immutableProperty: Int = 100 // 불변 프로퍼티
4  
5  static var typeProperty: Int = 100 // 타입 프로퍼티
6  
7  // 인스턴스 메서드
8  func instanceMethod() {
9    print("instance method")
10  }
11  // 타입 메서드
12  // 재정의 불가 타입 메서드 - static
13  static func typeMethod() {
14    print("type method -static")
15  }
16  // 재정의 가능 타입 메서드 - class
17  class func classMethod() {
18    print("type method -class")
19  }
20}
