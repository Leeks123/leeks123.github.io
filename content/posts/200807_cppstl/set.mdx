수학에서의 집학과 같은 개념
중복이 없는 항목들이 모인 것
(중복 허용하려면 MultiSet)

순서가 필요 없고, 고유 값을 원하는 경우 최선의 자료구조
데이터를 비순차적으로 저장할 수 있는 자료구조

중복된 값을 골라내야 할 때, 빠른 Look Up을 해야 할 때, 순서는 상관없을 때 사용

균형잡힌 이진트리. 노드기반 컨테이너
원소 삽입과 삭제, 탐색 등의 연산은 O(logn)을 보장
삽입 시 자동으로 정렬. 디폴트 정렬은 오름차순

연관 컨테이너 중 하나. key를 바탕으로 대응하는 값을 얻을 수 있는 구조

```cpp
#include <iostream>
#include <set>
using namespace std;
int main(){
    //int 자료형을 저장하는 균형잡힌 이진트리 생성
    set<int> s;
    
    //원소 삽입
    s.insert(5);
    s.insert(4);
    
    // 2번째 원소 삭제
    s.erase(++s.begin());
    
    //6값을 가지는 원소를 찾음 있다면 해당 원소의 주소값,  없다면 s.end()리턴
    if(s.find(6)!=s.end())
        printf("존재합니다.\n");
    else
        printf("없습니다.\n");
    
    //저장된 원소의 수를 리턴
    printf("%d\n", s.size());
    
    // 원소값에 해당하는 개수를 반환
    printf("%d\n", s.count(4)); // 있으면 1, 없으면 0
    
    //모든 원소 삭제
    s.clear();

    return 0;
}
```
